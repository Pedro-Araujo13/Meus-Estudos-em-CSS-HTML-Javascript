Whatsapp como CyberAtaque!

Descubra como os hackers estão usando o Whatsapp para hackear computadores!

Recentemente, estudiosos da área descobriram uma nova forma de cyberataque, os chamados crackers
enviam para o Whatsapp Web de computadores que possuem o sistema operacional Windows, um arquivo .zip,
e se caso o usuário baixar aquele arquivo e o executar, o vírus invade o computador, instala uma inicialização
(para evitar que o executável se encerre quando o computador for inicializado) e dessa forma, a máquina
fica sob comando do atacante.

Os casos ainda são recentes no mundo, porém, mais de 90% dos casos ocorrera, no Brasil, e têm como foco computadores
administrativo, o nome do ataque é chamado de 'Sorvepotel'.
